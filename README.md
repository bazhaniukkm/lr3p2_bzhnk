 Опис
Програма розв’язує нелінійне рівняння:
\[
f(x) = -x^4 - 5x^3 - 0.25x^2 + 2
\]
двома методами:
- Метод половинного ділення
- Метод хорд (секант)

Передбачено:
- текстове меню
- режим налагодження (вивід кожної ітерації)
- контроль ліміту ітерацій з вибором дії
- вимір часу обчислення
- перевірка похідної на проміжку
- пошук проміжку ізоляції

для звітц
Мета роботи
- Вивчити методи розв’язання нелінійних рівнянь.
- Реалізувати програму на мові С для знаходження коренів рівняння двома методами.
- Дослідити збіжність методів залежно від вибору початкових умов.
- Реалізувати режим налагодження, контроль кількості ітерацій, вимір часу виконання.
- Перевірити похідну та підтвердити наявність єдиного кореня на проміжку.
Вихідні дані
Функція варіанту 1:
f(x)=-x^4-5x^3-0.25x^2+2
Похідна:
f'(x)=-4x^3-15x^2-0.5x

Алгоритм
- Користувач обирає метод у текстовому меню.
- Вводить параметри: інтервал [a,b] або початкові наближення x_0,x_1, точність, максимум ітерацій, режим налагодження.
- Програма виконує обчислення:
- Метод половинного ділення: поступове звуження інтервалу.
- Метод хорд: побудова хорд і уточнення кореня.

5. 	Вимірюється час виконання.
6. 	Виводиться корінь, значення функції в корені, кількість ітерацій, час.
7. 	Додатково: перевірка похідної на проміжку та пошук проміжку ізоляції.
